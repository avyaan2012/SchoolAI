Student-only AI study website

Study from your real class materials.

ClassQuiz AI helps students practice for quizzes using school resources first. If there are not enough class materials, it recommends trusted web resources like videos, articles, exercises, simulations, flashcards, and practice quizzes.

This MVP uses demo data. Real Google Classroom/Schoology API connection can be added later.

How it works

  1. Sync your classes.
  2. Pick a class and quiz topic.
  3. Take an AI-generated practice quiz.
  4. See weak spots and get a study plan.
