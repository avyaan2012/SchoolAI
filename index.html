Student-only Google Classroom sync

Connect Google Classroom, read Slides, and generate AI questions.

This version has no demo fallback. It imports real Google Classroom materials, tries to read text from attached Google Slides, and uses Gemini to generate practice questions.

Before testing, edit config.js and paste your Google OAuth Web Client ID.

Real sync flow

  1. Click Connect Google Classroom.
  2. Approve Google permissions.
  3. The app imports your classes and materials.
  4. The app reads attached Google Slides when permissions allow it.
  5. Generate AI practice questions from the imported content.

No fake data

This version will not create sample classes. If Google sync fails, it shows the error.

Read-only access

It requests read-only Classroom scopes, so it does not edit or delete class content.

AI backend required

For Gemini AI generation, deploy on Vercel or connect this site to a Vercel API endpoint.
